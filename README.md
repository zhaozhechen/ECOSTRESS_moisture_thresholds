# ECOSTRESS Moisture Thresholds
This Repository includes codes for the manuscript "Increasing limitation of compound soil and atmospheric moisture stresses on evapotranspiration"

Detailed descriptions of datasets and methods can be found in the manuscript. 

## Data
All datasets used in this project are publicly available.

Main datasets include:
- [AmeriFlux dataset](https://ameriflux.lbl.gov/)
- [ECOSTRESS Level-3 DisALEXI-JPL Evapotranspiration](https://lpdaac.usgs.gov/documents/999/ECO3ETALEXI\_User\_Guide\_V1.pdf)
- [SMAP L4 root-zone soil moisture](https://nsidc.org/data/spl4smgp/versions/3)
- [ERA5 climate data](https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=overview)
## Quantification of moisture thresholds and sensitivity
`CONUS_Thresholds_0.25D` includes codes used to process datasets and calculate moisture thresholds and sensitivity.
## Drivers of spatial patterns in moisture sensitivity
`CONUS_RF` includes codes used to analyze drivers explaining the spatial patterns of moisture sensitivity, using random forest models.
## Frequency of ESI reduction under future climates
`CMIP6_Projection` includes codes used to explore how ESI reduction changes under future climate conditions.
